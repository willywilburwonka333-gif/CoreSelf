# Core Self

**Version:** Genesis 0.0.6  
**Founder:** Dylan Corr  
**First Core:** Dylan Core  

Core Self is a lifelong AI platform designed to help one human become the highest version of themselves.

## Start Web App

```bash
cd apps/web
npm install
npm run dev
```

Open forwarded port **5173** in Codespaces.
